# HRIS System

A modern Human Resource Information System (HRIS) built with Laravel and React.

## 📖 Overview

HRIS System is a web-based application that helps companies manage employee data, payroll processing, and payslip generation efficiently.

This project is developed as a portfolio project using modern technologies and software engineering best practices.

---

## 🚀 Features

- Authentication & Authorization
- Employee Management
- Department Management
- Position Management
- Payroll Processing
- Payslip Generation (PDF)
- Dashboard & Statistics

---
